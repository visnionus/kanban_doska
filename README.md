# 📋 KanbanFlow — Канбан-доска

Красивая и функциональная канбан-доска на чистом HTML/CSS/JS.

## ✨ Возможности

- 🎯 **Drag & Drop** — перетаскивание задач между колонками
- 🌓 **Тёмная/Светлая тема** — переключатель с сохранением выбора
- ✏️ **Создание/Переименование/Архивирование/Удаление** задач
- 🎨 **Цветовая подсветка** — 20 предустановленных цветов
- 😀 **Эмодзи для каждой задачи** — выбор из 30 иконок
- 🏷️ **Текстовые пометки** — дополнительные метки на карточках
- 📦 **Архив** — отдельная секция с возможностью восстановления
- 💾 **Сохранение в localStorage** — данные не теряются при перезагрузке
- ✨ **Анимации** — плавные переходы и эффекты
